## Unsubscribe Handling

Loops provides built-in unsubscribe handling via a footer link on every email.

- **Unsubscribe URL pattern:** `https://app.loops.so/unsubscribe/{contactId}/{mailingId}` (auto-generated by Loops in each email footer)
- **Behaviour:** Clicking the link immediately moves the contact to "unsubscribed" status; no further drip or transactional emails are sent
- **GDPR notes:**
  - One-click unsubscribe -- no confirmation page required
  - Loops retains contact record in "unsubscribed" state (not deleted) -- right-to-erasure requests require manual deletion via Loops dashboard or API (`DELETE /contacts/{id}`)
  - See `docs/integrations/loops.md` for API details
